# Weblog based on Astro Starter Kit: Blog

## Setup

Made using Bun.sh, but NodeJS probably works fine.

0. `git clone`
1. `bun install`
2. `bun run dev`

## Building your own

```sh
npm create astro@latest -- --template blog
```
